# My dotfiles

## Set up

### Backup dotfiles manually

- `.zshrc`, `.zshenv`

### Set up dotfiles

- Install and set up [Prezto](https://github.com/sorin-ionescu/prezto)
- Copy `.gitignore_global`, `.gitconfig`, `.vimrc` to `~/` if not exist

```console
git clone https://github.com/Sangen/dotfiles ~/.dotfiles
cd ~/.dotfiles
sh setup.sh
```
